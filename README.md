# Portfolio

Portfolio personal desarrollado para presentar mi perfil profesional, habilidades técnicas, experiencia y proyectos destacados en una experiencia web moderna e interactiva.

## Descripción

Este proyecto funciona como mi sitio de portafolio personal. Fue creado para mostrar de forma visual y ordenada quién soy, las tecnologías con las que trabajo, mi experiencia, algunos proyectos realizados y una vía de contacto directa.

La aplicación incluye una presentación inicial animada, una sección principal de bienvenida, información sobre mí, habilidades, proyectos, contacto y acceso a mi CV.

## Tecnologías utilizadas

Estas son las principales tecnologías usadas para crear el proyecto:

- Next.js 15
- React 18
- TypeScript
- Tailwind CSS
- Framer Motion


## Características

- Secciones organizadas para presentación profesional
- Animaciones e interfaz dinámica
- Listado de proyectos con recursos visuales
- Acceso al CV en formato PDF
- Estructura modular basada en componentes reutilizables

## Estructura del proyecto

```text
src/
  app/          # Configuración principal de la app y rutas
  components/   # Componentes reutilizables de la interfaz
  data/         # Datos de experiencia, skills y proyectos
public/
  img/          # Imágenes del portfolio
  PaulaSuarezCV.pdf
```



El objetivo de este portfolio es centralizar mi presentación profesional en un solo sitio, mostrando de manera clara mi perfil como desarrolladora, los proyectos en los que trabajo y las tecnologías que utilizo.
